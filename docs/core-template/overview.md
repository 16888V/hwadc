## Blogger version

Menggunakan template versi `3`:

{% highlight html %}
<html b:layoutsVersion='3'>...</html>
{% endhighlight %}

Menggunakan gadget versi `2`:

{% highlight html %}
<html b:defaultwidgetversion='2'>...</html>
{% endhighlight %}

{% highlight html %}
<b:widget version='2'>...</b:widget>
{% endhighlight %}
